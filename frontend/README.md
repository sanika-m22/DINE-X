# DineX - Gourmet Restaurant Experiences

A modern, visually appealing frontend for finding the best restaurants based on user preferences, rebranded and redesigned with an organic, gourmet aesthetic.

## Tech Stack
- **React 19**
- **Vite 8**
- **Tailwind CSS 4**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)
- **React Router 7**
- **Leaflet & React-Leaflet** (Map View)

## Features
- **Organic Landing Page**: Inspired by gourmet food menus with organic shapes and elegant serif typography.
- **Preferences Form**: Tailored form to filter by budget, cuisine, and rating with a cream/sage aesthetic.
- **Results Grid & Map**: 
    - Premium card layout with match scores and "HOT" labels.
    - Integrated Map View using OpenStreetMap and custom markers.
- **Branding**: Sophisticated **DineX** identity using sage green, terracotta, and cream palettes.

## Getting Started

1. Navigate to the project directory:
   ```bash
   cd restaurant-recommender
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) (or 5174) in your browser.
